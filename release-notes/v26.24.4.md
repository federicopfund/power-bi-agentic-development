# v26.24.4

`pbir-cli` (reports plugin): `pbir desktop list` is now the mandated first
command whenever the user mentions Power BI Desktop:

- General workflow step 1: when Desktop is mentioned or the report is said
  to be open there, run `pbir desktop list` before anything else; it maps
  each running instance to its open file (locating the report on disk) and
  confirms the bridge works before edits begin
- Exploration command block now leads with `pbir desktop list`
- Explicit platform note: `pbir desktop` is Windows-only; on macOS and Linux
  it must not be used (every invocation fails); verify through
  `pbir validate --all` and sandbox publishing instead
- Versions: reports plugin, pbir-cli skill, and marketplace metadata bumped
  to 26.24.4
